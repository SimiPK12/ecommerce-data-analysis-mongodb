Dataset Brief

This dashboard is built on the Olist Brazilian e-commerce dataset from Kaggle. It combines customers, orders, order items, payments, reviews, products, sellers, geolocation, and category translation data to uncover patterns in buyer behavior, revenue, logistics, and product performance.
